Personalization Semantics

Personalization Semantics Task Force

Goal

Developing support for personalization of content on the web.

Authoring/Bootstrapping

  • The approach needs input from authors.
  • We aim to make this as easy as possible.
  • We've seen things like <main> work very well, because there is wide potential applicability/neatness for developers.
  • If we can show the potential gains for users, we can create demand…
  • …and the sorts of things we’re expressing are quite fundamental.

W3C Personalization Task Force

  • Part of the Accessible Platform Architectures Working Group
  • …within the Web Accessibility Initiative
  • …of the World Wide Web Consortium.
  • More importantly: part of your web!

This is BIG

  • Multiple stages to the plan
    • Personalization Semantics Explainer
    • Module 1 Adaptable Content (nearing wide community review)
    • Module 2 Adaptable Help and Support
    • Module 3 Adaptable Tools
  • Years in the making, and to come—get in on the ground floor!

This is REALLY BIG

  • Focus on cognition, cognitive disabilities and usability.
    • Related: Making Content Usable for People with Cognitive and Learning Disabilities
  • Universally applicable (to varying degrees).
  • Could help make the web a lot more mailable and adaptive for people.
  • Built on simple and robust tech, so has a good chance!
  • …with much room to grow later on.

Personalization Semantics Module 1

  • Programmatically identify the purpose of various interface elements.
  • Support simplification of user interfaces and content.
    • Showing what’s important.
    • Removing distractions.
  • Support presenting symbols instead of words.
  • …and more!

Action, Destination and Purpose attributes

  • <button action="checkout">Check out</button>
  • TODO: check <a href="..." destination=“help”>Help</a>
  • <input type="text" purpose="home tel" />

  • Note: we are using data-* attributes for now.

  • Anticipate many different adaptations/user agents/extensions, for different user groups and cases.
